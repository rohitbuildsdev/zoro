# ZORO

ZORO is an offline-first Personal Operating System currently being developed in Python.

Unlike traditional chatbots, ZORO is designed to help users execute goals, manage tasks, and eventually answer a single question:

"What should I do right now?"

## Current Features (v0.1.0)

* Faster-Whisper Speech Recognition
* Wake Word Detection
* Intent Parsing
* Command Routing
* Application Launching
* Website Launching
* Text-to-Speech Responses
* Offline Operation

## Architecture

Speech Input
→ Whisper STT
→ Wake Word Detection
→ Intent Parser
→ Router
→ Action Execution
→ TTS Response
